DisasterNet — India Disaster Relief AI

PL_GENESIS · FRONTIERS OF COLLABORATION HACKATHON
DisasterNet is an offline-first, multi-role disaster coordination platform built for India's national disaster response ecosystem. It enables real-time collaboration between field rescue workers, hospital staff, victims in distress, and central command stations — even when internet connectivity fails — by combining a browser-based mesh network, AI-powered triage, and decentralized infrastructure.
🎯 PROBLEM
India's disaster response suffers from: siloed data across agencies, connectivity blackouts in affected zones, delayed compensation to victims, and no unified command layer for multi-state events involving NDRF, SDRF, hospitals, and NGOs simultaneously.
✅ SOLUTION
A single platform unifying 4 operator roles, AI triage via Impulse (Anthropic Claude), FHE-encrypted medical records (Zama TFHE), offline mesh via BroadcastChannel, IPFS field record pinning, and automated on-chain compensation triggers via Starknet.
🏗 ARCHITECTURE — SPONSOR INTEGRATIONS
⚡ Impulse AI (Anthropic Claude)
Streaming SSE triage agent — field reports → zone-specific routing commands in real-time. Anticipatory alert engine predicts criticality before events escalate.
🔐 Zama TFHE
FHE encryption of all sensitive medical/personal records client-side in WASM before mesh broadcast. Lit Protocol key gate for hospital-only decryption.
📦 Storacha / Filecoin / IPFS
Every rescue log, SOS event, and compensation case is pinned to IPFS for immutable provenance. Offline queue syncs when connectivity restores.
🌍 NEAR Protocol
Volunteer identity anchoring. Rescue workers verified on-chain via NEAR; Hypercerts minted per mission for verifiable impact records.
💰 Starknet (Sepolia testnet)
Smart contract on Starknet Sepolia manages compensation disbursement. RPC reads are live (no wallet needed). Connect ArgentX or Braavos in the Funds tab to submit real on-chain disbursement transactions.
🌐 World ID (Worldcoin)
Sybil-resistant volunteer verification — prevents duplicate volunteer registrations across relief operations.
